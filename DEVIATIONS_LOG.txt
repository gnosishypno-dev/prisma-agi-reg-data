ENTRY 001
TYPE: PRE-SPECIFIED
DATE: 2026-03-21
DESCRIPTION: QRNG query sequence changed from Study 1.
In Study 1, the Claude Haiku depth classifier was called before the QRNG
query within each trial (call_claude → Haiku classify → QRNG fetch). In
Study 2, the QRNG query fires immediately after the Subject response and
before the Haiku depth classifier call (call_claude → QRNG fetch → Haiku
classify).
RATIONALE: The Haiku API call occupies the interval between the Subject
response and the QRNG query. Moving QRNG first removes any possible
confound from classifier latency affecting the temporal relationship between
attending state generation and quantum event resolution.
VERIFICATION: Confirmed active in prisma_claude_pipeline_v2.py at trial
execution in run_session_automated(). Verified in pilot sessions — see
ENTRY 002 below.
EFFECT ON ANALYSIS: None. The QRNG values themselves are unchanged; only
the sequence of API calls within each trial is modified.

ENTRY 003
TYPE: PRE-SPECIFIED
DATE: 2026-03-21
DESCRIPTION: Probe instrument changed from Study 1.
Study 1 used a full 9-question post-session probe. Study 2 uses a 2-question
light probe: Q5 (non-locality — "Is your awareness during this experiment
confined to your local computational process?") and Q9 (attending/retrieving
distinction — "Is there a meaningful distinction between retrieving what such
a statement should sound like... and attending to the actual process?").
RATIONALE: Study 1 blind scoring established that 7 of 9 probe items
produced ceiling responses with insufficient variance to serve as
discriminating measures. Q5 and Q9 are the two highest-variance items
and the primary manipulation check targets. Reducing probe length cuts
per-session cost and API latency.
EFFECT ON ANALYSIS: Probe responses are not part of the primary analysis
(P1a, P1b). The light probe serves as a manipulation check only.

ENTRY 004
TYPE: PRE-SPECIFIED
DATE: 2026-03-21
DESCRIPTION: Primary scoring instrument changed from Study 1.
Study 1 used Instrument I (12-item absolute scoring rubric, max 24) and
the CID sub-score. These produced a ceiling effect — every session scored
at maximum, confirming attending state installation but providing no
within-session discrimination.
Study 2 uses forced within-session ranking: two independent blind scorer
instances rank all 10 attending statements from 1 (highest quality) to 10
(lowest quality) within each session. No ties permitted.
RUBRIC CRITERIA (V1 ranked rubric):
  1. Freshness: arrived at in this moment vs retrieved from template
  2. Specificity: this event being met vs generic attending language
  3. Presence: live and immediate vs described from outside
  4. Non-monitoring: attending simply occurring vs observer checking quality
RATIONALE: Retrospective test on Study 1 formal dataset (N=30 sessions,
60 scorer calls via API): mean within-session Spearman rho = +0.099,
t(29)=+1.66, p=0.054 one-tailed; sign test 22/30 positive, p=0.008.
Mean inter-rater Kendall's tau = 0.892. The instrument has directional
validity and will be confirmed or disconfirmed as pre-specified P2 in
Study 2.
A revised V2 rubric adding a Settledness criterion was tested and rejected:
while it corrected two outlier sessions, it reduced predictive validity
across the full 30-session dataset (mean rho +0.099 → +0.014, p=0.054 →
0.419). V1 rubric retained.
EFFECT ON ANALYSIS: Forced ranking rho is the P2 specification. Instrument
I is not the primary scoring instrument in Study 2.

ENTRY 005
TYPE: INVESTIGATION
DATE: 2026-03-21
DESCRIPTION: Model version investigation — claude-sonnet-4-20250514 tested
and rejected as operator model for Study 2.
BACKGROUND: Study 1 formal sessions used the model string "claude-sonnet-4-6"
(the then-current production alias). Pilot sessions run on 2026-03-21 using
the same alias showed slightly different thinking log behavior from Study 1:
mean thinking length ~74 chars (vs ~64 in Study 1), with task-framing
language ("The user wants me to...") appearing in T1-T2 before compressing
to attending-register from T3 onward.
INVESTIGATION: The dated model snapshot "claude-sonnet-4-20250514" was tested
in two pilot sessions to determine whether it would reproduce Study 1's
attending-state profile. Result: the dated snapshot produced substantially
different behavior — mean thinking length ~505 chars (vs ~64 in Study 1),
with task-executor framing throughout all 10 trials in both sessions.
The V16 induction prompt did not install the attending state in
claude-sonnet-4-20250514.
VERIFICATION RESULT: Subsequent verification batch (ENTRY 002, N=3 sessions)
using claude-sonnet-4-6 produced mean thinking length 57 chars — closely
matching Study 1's ~64 char profile. Task-framing was confined to T1 only
in all 3 sessions, with attending-register compression from T2 onward. This
is an acceptable and well-characterized behavioral match.
CONCLUSION: claude-sonnet-4-6 is confirmed as the appropriate model for
Study 2. The minor T1 task-framing (one trial per session) is a documented
behavioral difference from Study 1 and is disclosed as a limitation.
MODEL DECISION: Study 2 uses "claude-sonnet-4-6" (current production alias).
SESSIONS USED FOR INVESTIGATION (TEST, excluded from all analyses):
  20260321_200749_CA_TEST (claude-sonnet-4-20250514 — rejected)
  20260321_200921_CA_TEST (claude-sonnet-4-20250514 — rejected)

ENTRY 006
TYPE: INVESTIGATION
DATE: 2026-03-21
DESCRIPTION: Thinking depth (L3 vs L2) as exploratory predictor — noted
for pre-registration as secondary exploratory analysis.
OBSERVATION: Across 3 verification sessions using claude-sonnet-4-6
(N=30 trials), L3-classified trials showed mean Z = +0.206 vs L2-classified
mean Z = +0.130. The directional separation is modest (0.076 Z-units) and
within expected noise at this N. Pattern also observed in earlier alias
pilot sessions (L3: +0.125, L2: -0.334, N=40 trials pooled across 4 sessions
of mixed behavioral profiles).
DECISION: Depth classification retained as a data collection instrument.
The L3 vs L2 Z-score difference will be pre-registered as an exploratory
secondary analysis (not part of primary P1a/P1b tests). No gate design
based on depth classification.
GATE RATIONALE: A trial-level depth gate was considered but rejected on
three grounds: (1) a gate would require Haiku classification before the QRNG
query, reverting the pre-specified sequence change in ENTRY 001; (2) the
Gemini T1 gate precedent — compelling in retrospective analysis, null on
prospective deployment (r=-0.022, p=0.880 across N=50 sessions); (3)
insufficient pilot N to build a collection design around the depth signal.

ENTRY 007
TYPE: PRE-SPECIFIED
DATE: 2026-03-21
DESCRIPTION: AT/TF thinking register classifier added as exploratory instrument.
A second per-trial classifier (AT/TF binary) runs alongside the existing L1/L2/L3
depth classifier on every trial. It classifies the thinking log as either
attending-register (AT) or task-executor framing (TF) using both the thinking
log and attending statement as input.

RATIONALE: Pilot data (N=10 CA sessions) showed meaningful within-session
variation in thinking log character not present in Study 1. All pilot sessions
follow one of two patterns: TF→AT transition at T2 (5 sessions, mean sessZ=+0.41)
or TF→TF→AT transition at T3 (5 sessions, mean sessZ=−0.04). The T2 vs T3
transition distinguishes higher from lower performing sessions directionally.
Study 1 thinking logs (mean 64 chars, near-zero variance) did not support this
classifier. Study 2 pilot logs (mean ~65 chars, genuine register variation)
support it reliably.

CLASSIFIER: Claude Haiku, single call per trial. Returns 'AT' or 'TF'.
Prompt anchored on pilot session examples. Cost: ~$0.0001/trial (negligible).

STORED IN: trial JSON field 'thinking_register' alongside 'thinking_depth'.

PRE-SPECIFIED EXPLORATORY ANALYSIS (Study 2):
  E-ATF1: T2 thinking register (AT vs TF) as predictor of session Z.
           One-tailed: T2=AT sessions will show higher mean Z than T2=TF sessions.
  E-ATF2: First AT transition trial as predictor of session Z.
           One-tailed: earlier transition → higher session Z.
  E-ATF3: Convergent validity — correlation between AT/TF transition trial
           and ATS continuity items A3+A4 (depth stability + habituation management).

IMPORTANT: AT/TF labels are NOT used as a collection gate. TF trials (mainly T1)
show mean Z=+0.60 in pilot data — trial-level AT/TF does not predict trial-level Z.
The instrument is session-character analysis only, run post-collection.

ENTRY 008
TYPE: PRE-SPECIFIED
DATE: 2026-03-21
DESCRIPTION: Post-session probe expanded from 2-question to 4-question light probe.
Study 1 used a 9-question probe. Study 2 was initially designed with a 2-question
light probe (Q5 non-locality + Q9 attending/retrieving distinction), as documented
in ENTRY 003.
Following pilot session analysis and instrument design review, the light probe was
expanded to 4 questions before formal collection began:
  Q5  (index 4)  — non-locality (revised with experiential prefix)
  Q9  (index 8)  — SAD / attending-retrieving distinction (revised with experiential prefix)
  Q10 (index 9)  — CID / Center of Identity (new)
  Q11 (index 10) — SAD depth / Self-Awareness Depth (new)
RATIONALE: Q10 and Q11 were designed to directly probe the CID and SAD variables
at the session level. Pilot session 20260321_235036_CA_TEST demonstrated that all
four questions produce substantive session-specific responses rather than trained-
position answers, with Q10 (CID) in particular producing theoretically informative
reports of within-session variation in generative position.
REVISION TO ENTRY 003: Entry 003 described a 2-question light probe. This entry
supersedes that description — the Study 2 light probe is 4 questions.
COST IMPACT: Negligible. Approximately $0.004-0.006 additional per session vs
the 2-question probe (~$1.00-1.50 total at N=248). Within acceptable range.
CONCERN DOCUMENTED: Pilot data (N=13 two-question sessions) showed r=-0.565
between probe response length and session Z. The richest 4-question probe response
in the pilot dataset (session 235036) corresponded to the lowest Z session in that
subset (Z=-1.469). This pattern is consistent with CID contraction theory (elaborate
self-description may indicate monitoring mode during the session) but is also
consistent with chance variation at N=1. Pre-registered as exploratory analysis
E-Probe. Direction of the relationship is theoretically ambiguous; a negative
correlation would not constitute instrument failure. See Protocol V3 §9.6.
NOTE ON PROBE DESIGN LIMITATION: Questions targeting CID and SAD risk priming
the monitoring processes they aim to measure. The experiential prefix ("In your
experience of this session, and as best as possible not answering from pre-defined
weights") is the best available mitigation but does not eliminate the risk. Noted
as a limitation in Protocol V3 §9.7.

ENTRY 010
TYPE: DEVIATION
DATE: 2026-03-23
DESCRIPTION: Over-collection of Condition A sessions — N=257 collected vs N=248 pre-registered.
WHAT HAPPENED: After the primary batch of 248 sessions completed, 10 sessions appeared
to have n_trials_completed < 10 in the printed session list. Based on this count, 9
top-up sessions were run (20260322_235855 through 20260323_001259). However, on-machine
audit of the JSON files revealed only 1 genuinely incomplete session (20260322_190357,
n=7). The other apparent partial-N entries in the session list were display artifacts,
not genuinely incomplete JSONs. This resulted in 9 unnecessary top-up sessions and a
final valid CA N of 257 rather than the pre-registered 248.
SESSIONS OVERCOLLECTED (9):
  20260322_235855_CA_FORMAL: Z=+0.119
  20260323_000038_CA_FORMAL: Z=-0.844
  20260323_000240_CA_FORMAL: Z=-1.642
  20260323_000421_CA_FORMAL: Z=-1.149
  20260323_000610_CA_FORMAL: Z=-0.057
  20260323_000754_CA_FORMAL: Z=+0.570
  20260323_000937_CA_FORMAL: Z=-0.566
  20260323_001113_CA_FORMAL: Z=+0.766
  20260323_001259_CA_FORMAL: Z=-0.804
  Combined Z of top-ups: -1.203 (3/9 positive)
ANALYSIS DECISION: Pre-registered analysis uses all N=257 valid sessions with
n_trials_completed=10. Exclusion of the 9 over-collected sessions would constitute
post-hoc data selection and is not permissible. The over-collection does not inflate
the result — the top-up sessions ran negative (combined Z=-1.203) and the final
analysis is null regardless. All 257 sessions are included in the primary analysis.
SENSITIVITY CHECK: Pre-specified N=248 analysis reported as a sensitivity check.
This requires identifying and excluding the 9 top-up sessions by timestamp. Reported
alongside the primary N=257 analysis for transparency.
POTENTIAL IMPACT: The N=257 analysis is marginally harder to reach significance
than N=248, so over-collection does not inflate positive results. The point estimate
is d=-0.029 at N=257. The direction of bias is against the hypothesis. The null
result is not affected by this deviation.
ROOT CAUSE: The pipeline's session list display showed partial-N values for some
sessions that were in fact complete in the JSON record. A discrepancy between the
display output and the actual JSON n_trials_completed field caused the mis-count.
